# SSBT Campus Navigation

A premium, high-performance campus navigation system for SSBT COET Jalgaon. Built with Google Maps JavaScript API and a custom "Liquid Glass" design system.

## ✨ Features
- **Google Maps Integration**: Accurate satellite and roadmap views.
- **Walking Directions**: Turn-by-turn guidance across the campus.
- **Liquid Glass UI**: Apple-style glassmorphism with GPU acceleration.
- **Precise Locations**: Over 20+ verified campus departments and facilities.

## 🚀 Deployment
- Hosted via GitHub Pages / Netlify.
- Restricted API Keys for maximum security.

## 🛠️ Built With
- HTML5, CSS3 (Vanilla), JavaScript (ES6)
- Google Maps Platform
